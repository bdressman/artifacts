# artifacts
Scripts to play the Artifacts MMO game at https://www.artifactsmmo.com/

## Setup
1. Clone the repository
2. Install dependencies:

```
npm install
```

3. Copy the example environment file:

```
cp .env.example .env
```

(On Windows, just copy and rename the file.)

4. Edit `.env` and add your Artifacts API token.

5. Run the project:

```
npm start
```
